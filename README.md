### 🔨 What I'm Working On

**[Vixxon](https://vixxon.online)** — a full-stack social platform in production with ~50 live users
- Real-time WebSocket chat, XP/streaks, voice notes, and anonymous Q&A
- Django, Django Channels, Celery, Redis, PostgreSQL + PgBouncer, HTMX, Bootstrap 5
- Deployed on a hardened cloud VPS (Nginx, Daphne, systemd, Fail2ban, UFW, SSL)
- Cloudflare R2 for media storage, PostHog + Sentry for analytics and monitoring
- CI/CD pipelines planned next

**AI Coding Assistant** — in active development
- Combines local LLMs with Google Gemini GenAI via LangChain tool-calling agents
- Retrieval-Augmented Generation (RAG) for context-aware code understanding
